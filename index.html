Section 03

Core Modules

Four independent modules — each designed to do one job perfectly.

Module 01 · Optimize

Connection Optimizer

Tunes how your connection negotiates with servers for a more stable, consistent link — selectively, leaving everything else at native speed.

  • Dashboard — engine start/stop, health status, network detection
  • Profile list — your own targets or the bundled default set
  • Network Analysis — Quick or Deep Scan + adaptive engine
  • Auto-failover — rotate profiles when quality drops
Module 02 · Tunnel

Private Discord

A Discord client running through its own Tor privacy network — fully separated from the main install. Isolated session.

  • New account setup — store credentials locally only
  • Use saved account — load identity with one click
  • Tunnel-establishing indicator with live status
  • Forget saved credentials anytime
Module 03 · Verify

File Integrity

Compares your installation against original server files and repairs corrupted components.

  • Connecting → Downloading → Extracting
  • Scanning → Repairing → Cache cleanup
  • Live report on every repaired file
  • 100% integrity confirmation when clean
Module 04 · Resolve

Encrypted DNS

DNS-over-HTTPS via a local dnscrypt-proxy — your ISP can't see or hijack what you resolve. One toggle on the main screen.

  • One-click DoH (DNS-over-HTTPS) on/off
  • Providers: Cloudflare, Google, Quad9, AdGuard
  • Reset to automatic (DHCP) anytime
  • Crash-safe — your real DNS is restored on next launch

Usage

No complex setup. Download, install, use. Follow the steps.

01

Download & Install

Grab the installer from GitHub, double-click, follow the wizard.

02

Pick Your Language

Choose between Türkçe, English or Русский on first launch.

03

Everything On One Screen

The app opens to a single dashboard — your ISP and best profile are detected automatically. DNS, analysis and Discord are all one click away.
